## Overview
This repository contains a 3D CAD design of a suspension shock absorber, created for mechanical visualization, prototyping, and educational use.  
The model represents a realistic coil-over shock absorber commonly used in automotive, robotics, and mechanical systems.

## Design Features
- Realistic coil spring geometry  
- Shock body and internal piston structure  
- Upper and lower mounting joints  
- Clean and compact mechanical design  
- Suitable for animation and assembly studies  

## Applications
- Mechanical engineering projects  
- Robotics suspension systems  
- Automotive concept designs  
- CAD learning and practice  
- Motion and stress analysis  

## Files Included
- 3D CAD model files  
- Assembly files  
- Render-ready geometry  
- Editable source design files  

## Software Compatibility
- Onshape  
- Blender (for animation and rendering)  
- Fusion 360  
- SolidWorks (via compatible export formats)
